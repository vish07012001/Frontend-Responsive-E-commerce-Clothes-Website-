# Frontend-Responsive-E-commerce-Clothes-Website
# Description
This project is a responsive e-commerce product display webpage built using HTML,CSS, JavaScript, Bootstrap 5, and Font Awesome for icons. It features a fully functional product carousel with smooth navigation, product card displays with hover effects, and an interactive product quantity selector. The design is optimized for different screen sizes, ensuring a seamless user experience on both desktop and mobile devices.

# Key Features
 <ul>
<li>Responsive Navbar: A custom-designed, responsive navigation bar that adapts to various screen sizes, including mobile. It includes social icons, contact links, and product categories. </li>
<li>Product Carousel: A dynamic image carousel for showcasing products with navigation controls and text overlay for promotional banners. </li>
<li>Interactive Product Cards: Each product is displayed in a card format with hover effects showing options to view, add to wishlist, or compare. </li>
<li>Quantity Selector: Users can adjust product quantities using increment and decrement buttons for easy selection. </li>
<li>Responsive Design: The layout is designed to be fully responsive, adjusting to screen sizes from large desktop monitors to mobile devices. </li>
<li>>Sidebar Filters: Users can filter products by category, size, and color using checkboxes and radio buttons. </li>
<li>Price Range Slider: An interactive slider (noUiSlider) allows users to filter products by price.</li>
<li>Product Grid: Responsive product cards with hover effects for adding items to the cart, wishlist, or comparing them. </li>
<li>Pagination: Navigate through multiple pages of products easily. </li>
</ul>
